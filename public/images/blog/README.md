## Image Generation

When adding new blog posts:
1. Generate image using create-post.js or Google ImageGen
2. Save with format: {blog-slug}.jpg
3. Recommended dimensions: 1200x628px (Open Graph standard)
4. Add reference to blog post object in posts.ts
